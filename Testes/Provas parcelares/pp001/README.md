# Programação I

## 📖 Sobre

Este repositório contém a resolução da 2º Prova Parcelar da **turma M3** disciplina de **Programação I**. 

---

## 📝 Descrição

A prova reúne exercícios que exigem o desenvolvimento de soluções utilizando a linguagem C, envolvendo estruturas de controle, manipulação de dados, vetores, matrizes e cadeias de caracteres. 

Os exercícios foram implementados buscando clareza, organização e eficiência.

---

## 🎯 Objetivos

- Praticar a resolução de problemas utilizando a linguagem C.
- Desenvolver o raciocínio lógico e algorítmico.
- Aplicar estruturas de decisão e repetição.
- Trabalhar com vetores, matrizes e strings.
- Fortalecer os conhecimentos fundamentais de programação.

---

## 📚 Conteúdo Abordado

- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do...while`)
- Vetores
- Matrizes
- Manipulação de strings
- Operações com caracteres
- Entrada e saída de dados (`scanf` e `printf`)
- Lógica de programação
- Desenvolvimento de algoritmos

---

## 🛠️ Tecnologias

- Linguagem C
- Embarcadero Dev-C++
- Visual Studio Code
