# 📘 Exercício 02

## 📖 Descrição

Este exercício tem como objetivo reforçar os conceitos básicos da linguagem C, explorando estruturas fundamentais de programação, como entrada e saída de dados, operadores, expressões e lógica computacional.

O programa recebe os dados solicitados pelo utilizador, realiza os cálculos ou operações exigidas pelo enunciado e apresenta os resultados de forma clara e organizada.

---

## 🎯 Objetivos de Aprendizagem

- Praticar a leitura de dados pelo teclado;

- Aplicar operadores aritméticos e lógicos;

- Desenvolver o raciocínio algorítmico;

- Produzir saídas formatadas utilizando a linguagem C.

---

## 🛠️ Tecnologias Utilizadas

-Linguagem C

## 📚 Conceitos Trabalhados

- Entrada e saída de dados (scanf e printf);

- Organização básica de programas em C.
