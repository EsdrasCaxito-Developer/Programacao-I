# 📘 Exercício 10

## 📖 Descrição

**Área de uma circunferência**

Desenvolva um programa em linguagem C que calcule propriedades de uma circunferência com
base no valor do raio fornecido pelo utilizador via linha de comando. O programa deve solicitar
ao utilizador a introdução de um valor real correspondente ao raio e, em seguida, calcular e
apresentar:

- A área da circunferência;
- O perímetro (comprimento da circunferência).

O valor de π (PI) deve ser definido através de uma diretiva do pré-processador (#define) num
ficheiro de cabeçalho separado, o qual deverá ser incluído no programa principal e usar a biblioteca math.h para elevar o raio ao quadrado. As fórmulas a utilizar são:

A = π × r^2

P = 2 × π × r

---

- Desenvolver uma solução utilizando a linguagem C.

- Praticar estruturas básicas da linguagem.
