# 📘 Exercício 12

## 📖 Descrição

**Área de um paralelepípedo retangular**

Escreva um programa em C que leia as dimensões a, b e c de um paralelepípedo retangular e,
em seguida, calcule e imprima a sua área. A fórmula para calcular a área é:

A = 2(ab + ac + bc)


---

## 🎯 Objetivos

- Interpretar corretamente o enunciado do problema.

- Desenvolver uma solução utilizando a linguagem C.

- Praticar estruturas básicas da linguagem.

- Consolidar conhecimentos de lógica de programação.
