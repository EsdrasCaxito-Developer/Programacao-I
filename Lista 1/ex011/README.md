# 📘 Exercício 11

## 📖 Descrição

**Inversão de um número**

Escreva um programa em C que leia um inteiro consistindo de três dígitos como entrada e o
exiba invertido.

Exemplo: se a entrada for 123, 321 é exibido.

---

## 🎯 Objetivos

- Interpretar corretamente o enunciado do problema.

- Desenvolver uma solução utilizando a linguagem C.

- Praticar estruturas básicas da linguagem.

- Consolidar conhecimentos de lógica de programação.

---
