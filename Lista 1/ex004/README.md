# 📘 Exercício 04

## 📖 Descrição

**Parte fraccionária**

Escreva um programa C que leia um número real e imprima a sua parte fracionária. 

Exemplo: a parte fracionária de 3,09 é 0,09.

---

## 🎯 Objetivos

- Interpretar corretamente o enunciado do problema.

- Desenvolver uma solução utilizando a linguagem C.

- Praticar estruturas básicas da linguagem.

- Consolidar conhecimentos de lógica de programação.

---

## 🛠️ Tecnologias Utilizadas

Linguagem C
