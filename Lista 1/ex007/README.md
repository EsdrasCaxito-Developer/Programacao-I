# 📘 Exercício 07

## 📖 Descrição

** Distância entre 2 pontos ** 

Escreva um programa que calcule e imprima a distância dist (tipo double) entre dois pontos A e
B do plano cartesiano, cujas coordenadas (XA, YA) e (XB, YB) são inseridas no teclado. 

Usando a fórmula: dist = √(XA − XB)^2 + (Y A − Y B)^2

---

## 🎯 Objetivos

- Interpretar corretamente o enunciado do problema.

- Desenvolver uma solução utilizando a linguagem C.

- Praticar estruturas básicas da linguagem.

- Consolidar conhecimentos de lógica de programação.
