## 📖 Conteúdo das Listas 

### 📘 Lista 1 
— Fundamentos da Linguagem C Nesta lista são abordados os conceitos introdutórios da linguagem C, incluindo: 
- Entrada e saída de dados 
- Operações aritméticas 
- Expressões matemáticas 
- Conversão de unidades 
- Diretivas do pré-processador (`#define`) 
- Bibliotecas padrão (`stdio.h`, `stdlib.h`, `math.h`, `limits.h`, `float.h`) 
- Manipulação de números inteiros e reais 
- Geometria e cálculos matemáticos 

A lista contém **14 exercícios**. 

--- 

### 📗 Lista 2 
— Estruturas Condicionais e de Repetição 

A segunda lista introduz estruturas de decisão e repetição através de problemas envolvendo: 
- Condições (`if`, `else`) 
- Estruturas `switch` - Laços `while`, `for` e `do...while` 
- Validação de dados 
- Números primos 
- Números perfeitos 
- Números de Armstrong 
- Palíndromos 
- Sequências matemáticas 
- Geração de números aleatórios 

A lista contém **15 exercícios**. 

--- 

### 📙 Lista 3 — Vetores, Matrizes e Ponteiros 
Nesta etapa são explorados conceitos mais avançados da linguagem C, incluindo: 
- Vetores 
- Matrizes 
- Matrizes transpostas 
- Operações matriciais 
- Algoritmos de ordenação 
- Pesquisa sequencial e binária 
- Ponteiros 
- Aritmética de ponteiros 
- Manipulação eficiente de memória 

A lista contém **11 exercícios**.
