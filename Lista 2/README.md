# 📚 Lista de Exercícios 2 - Programação I (Linguagem C)

Repositório com as soluções da **Lista de Exercícios 2** da disciplina **Programação I**, do **1.º Ano da Licenciatura em Engenharia Informática (EINF2 T1)**.

---

## 📖 Sobre

Esta lista de exercícios tem como objetivo desenvolver o raciocínio lógico, a resolução de problemas e a aplicação de estruturas fundamentais da linguagem C, abordando principalmente estruturas de decisão, estruturas de repetição, operações matemáticas e manipulação de caracteres.

Cada exercício está organizado em sua própria pasta, contendo o código-fonte e um README explicativo.

## 🎯 Objetivos de aprendizagem

Ao concluir esta lista, o estudante será capaz de:

- Praticar programação estruturada em C.

- Desenvolver algoritmos utilizando estruturas condicionais.

- Aplicar estruturas de repetição (for, while e do...while).

- Trabalhar com operações matemáticas e lógica computacional.

- Resolver problemas envolvendo números, sequências e caracteres.

- Melhorar a capacidade de análise e implementação de algoritmos.
